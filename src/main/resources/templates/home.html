This is a web service to manipulate the data of covid19

The methods that can be called are the followings:

  • /summary/{name} to get the covid-19 summary data of a country where {name} is a path variable for the country name
  • /createSummary to create an entry for a country data
  • /updateSummary to update a country value(s)
  • /deleteSummary/{name} to delete a country havaing given {name}
